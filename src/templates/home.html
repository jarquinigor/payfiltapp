Bienvenido a PayFilt

PayFilt es un sistema web inteligente diseñado para la detección de fraude en operaciones digitales con tarjetas de crédito y débito.

Este sistema web brinda desde servicios de detección de fraude en línea para operaciones digitales con tarjeta y permite la detección de fraude en conjuntos de datos de operaciones digitales ya finalizadas. Asimismo, se brinda acceso a una sección de análisis en donde se pueden visualizar métricas definidas en el sistema para validar el rendimiento de las predicciones de fraude sobre los conjuntos de datos de operaciones digitales.

Detección en línea

Realice detecciones de fraude para operaciones con tarjetas de crédito en curso mediante el uso del servicio de detección web de PayFilt.

Detección en datasets

Realice detecciones de fraude en conjuntos de datos de operaciones con tarjetas de crédito ya finalizadas y evaluadas con el fines de diagnóstico.

Análisis en línea y datasets

Visualice métricas de análisis para la detección de fraude para operaciones con tarjetas de crédito en curso y en conjuntos de datos ya finalizadas y evaluadas.
